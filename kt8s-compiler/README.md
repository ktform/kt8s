## kt8s compiler

Interprets [image-dsl](../kt8s-image-dsl)'s and [kt8s-charts](../kt8s-charts)'s into dockerfiles and Kubernetes Yaml with [KSP](https://github.com/google/ksp).

Used alongside [kt8s-gradle-plugin](../kt8s-image-gradle-plugin), to propagate and manage build environment.
